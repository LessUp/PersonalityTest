# Changelog

## [Unreleased] - 2025-11-23

### Added
- **Frontend Animations**: Integrated `framer-motion` to add fade-in effects, staggered list animations, and interactive hover states for cards and buttons.
- **Documentation**: Added project root `README.md` with setup instructions and project overview.

### Changed
- **Font Loading**: Migrated from Google Fonts `@import` in CSS to `next/font/google` in `_app.tsx` for better performance and zero layout shift.
- **UI Interaction**: Enhanced hover effects on feature cards and discovery cards to be more tactile and responsive.
